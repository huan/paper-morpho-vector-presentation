# paper-morpheme-vector-presentation

A Thought of  Morphonological Vector Representation in Linguistic

## INSTALL

```shell
brew install pandoc pandoc-xxx
npm install -g nodemon
```

## BUILD

```shell
make clean
make build
```

The output files would be placed into `dist/` folder.
